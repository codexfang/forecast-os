# Forecast OS

A modern weather timeline and visualization dashboard that can search any city and view current conditions, 7-day forecasts, temperature trends, and optional historical data.

## Features

- **City Search** — Debounced search with geocoding autocomplete suggestions
- **Current Weather** — Temperature, feels-like, humidity, wind speed, and conditions
- **7-Day Forecast Timeline** — Horizontal-scrollable timeline on desktop, vertical stacked on mobile
- **Temperature Trend Chart** — Interactive line chart with optional precipitation bars (Recharts)
- **Historical Data** — Toggle to view past 14 days of weather data
- **Favorites System** — Save cities for quick access, persisted in localStorage
- **Unit Toggle** — Switch between Celsius and Fahrenheit instantly

## Tech Stack

- [React 19](https://react.dev)
- [Vite](https://vitejs.dev)
- [Recharts](https://recharts.org)
- [date-fns](https://date-fns.org)
- [Open-Meteo API](https://open-meteo.com)
